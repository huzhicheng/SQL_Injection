## 一个 SQL 注入的小样儿

1. SQL 脚本在 `resources/SQL`下，现在本地执行创建对应的数据库和表。

2. 修改 config.properties 中的数据库配置信息，包括数据库连接、账号、密码

3. 执行 App.java 启动。

4. 输入对应的参数即可开始测试了，比如 `Docker` 、`Docker' or 1=1 #` 等
